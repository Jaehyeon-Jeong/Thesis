# Stage 2 Reports

## English

Small report artifacts can be tracked here. Large predictions, checkpoints, and
raw data are excluded from GitHub.

Current small tracked artifacts:
- `data_audit/`: BTC OHLCV audit summaries.
- `tables/`: small planning/result tables such as split-count summaries.
- `figures/`: optional small selected report figures. Large figure folders
  should stay in Kaggle outputs.

## 한국어

작은 report artifact는 이 폴더에 둘 수 있습니다. 대용량 prediction, checkpoint,
raw data는 GitHub에서 제외합니다.

현재 tracking하는 작은 artifact:
- `data_audit/`: BTC OHLCV audit summary.
- `tables/`: split-count summary 같은 작은 planning/result table.
- `figures/`: 선택적으로 작은 report figure를 둡니다. 큰 figure folder는 Kaggle
  output에 남깁니다.
