# Stage 2 Grad-CAM Figures

## English

Small selected BTC Grad-CAM report copies can be placed here after full runs.
Large per-sample Grad-CAM folders are excluded from GitHub.

For the Stage 2 single-seed report, the required final figure is the
Figure-13-style best-configuration Grad-CAM:

- `stage2_single_seed_best_i60_ohlc_ma_vb_r20_gradcam_10perclass.png`
- `stage2_single_seed_best_i60_ohlc_ma_vb_r20_gradcam_10perclass_samples.csv`

This figure must contain `10` predicted-up BTC images and `10` predicted-down
BTC images. It is generated in Kaggle by
`notebooks/kaggle_stage2_best_gradcam_10_one_cell.md`.

## 한국어

Full run 이후 작은 BTC Grad-CAM report copy를 이 폴더에 둘 수 있습니다. 대용량
sample별 Grad-CAM folder는 GitHub에서 제외합니다.

Stage 2 single-seed 결과 보고에서 필요한 최종 figure는 Figure 13 스타일 best
configuration Grad-CAM입니다.

- `stage2_single_seed_best_i60_ohlc_ma_vb_r20_gradcam_10perclass.png`
- `stage2_single_seed_best_i60_ohlc_ma_vb_r20_gradcam_10perclass_samples.csv`

이 figure는 predicted-up BTC image 10개와 predicted-down BTC image 10개를 포함해야
합니다. Kaggle에서 `notebooks/kaggle_stage2_best_gradcam_10_one_cell.md`로
생성합니다.
