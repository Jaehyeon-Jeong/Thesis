# Stage 3 Configs

## English

Stage 3 config files will be added after `3-I1`.

The config layer should inherit Stage 2 data/split/metric settings and add only
Linear-adapter-specific settings such as adapter dimension, bias flag, selected
image windows, selected return horizons, selected image specs, and run seeds.

## 한국어

Stage 3 config file은 `3-I1` 이후 추가합니다.

config layer는 Stage 2 data/split/metric 설정을 상속하고, Linear adapter에 필요한
adapter dimension, bias flag, 선택 image window, 선택 return horizon, 선택 image spec,
run seed 같은 설정만 추가해야 합니다.
