# Stage 3 Notebooks

## English

Kaggle one-cell runners will be added after the Stage 3 implementation scripts
exist. The notebook wrappers should call shared `src/` and `scripts/` code rather
than duplicating model logic inside the notebook.

## 한국어

Stage 3 implementation script가 생긴 뒤 Kaggle one-cell runner를 추가합니다.
notebook wrapper는 model logic을 notebook 안에 중복 작성하지 않고, 공통 `src/`와
`scripts/`를 호출해야 합니다.
