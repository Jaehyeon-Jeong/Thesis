# N14-B2..B6 Derivatives/Leverage Conditional Bucket Analysis

## Purpose

This is a post-training bucket analysis for the N16 `ohlc_vb + funding + CFTC OI` context-FiLM model.
It uses predefined buckets only and does not train a new model.

## Main Interpretation

- The derivative/leverage context remains a small same-image improvement, not an overall best model.
- The model changes a limited number of decisions, so bucket-level evidence should be read as conditional diagnostics.
- The most defensible claim is that derivatives context sometimes acts as a bearish calibration signal for weak bullish chart predictions.
