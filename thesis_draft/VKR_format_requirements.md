## Confirmed Requirements

| Area | Confirmed rule | Source |
|---|---|---|
| Institution | The rulebook is for MIPT VKR preparation, defense, and storage. | ВКР regulation, section 1 |
| Language | The VKR is prepared in the language of instruction unless the GIA program defines another rule. | ВКР regulation, section 2.4 |
| Originality | Minimum originality: bachelor/diploma work at least 70%; master dissertation at least 80%. | ВКР regulation, section 2.15 |
| EBS upload | Final VKR text and annotation must be uploaded through the personal cabinet for plagiarism checking and EBS placement no later than 3 days before defense. | EBS order, section 2.2 |
| EBS identity | The uploaded work must be identical to the work submitted to the State Examination Commission. | EBS order, section 2.2 |
| EBS annotation | If the VKR is in English, the annotation may be in English. | EBS order, section 2.10 |
| Annotation length | Annotation must be no more than 1500 characters. | ВКР regulation, Appendix 1; EBS order, section 2.10 |
| Annotation content | Annotation should reflect goals, tasks, obtained results, and recommendations. | ВКР regulation, Appendix 1; EBS order, section 2.10 |
| Recommended length | Master dissertation: 50-70 pages excluding appendices. Diploma work: 40-50 pages excluding appendices. Bachelor work: 30-40 pages excluding appendices. | ВКР regulation, Appendix 1 |
| Required structure | Title page; annotation; table of contents; designations/abbreviations if needed; main text; list of used sources; appendices if needed. | ВКР regulation, Appendix 1 |
| Main text | Main text contains introduction, main content, and conclusion. | ВКР regulation, Appendix 1 |
| Introduction | Introduction should justify relevance, define object/subject, goals/tasks, methods, and briefly state theoretical/practical significance. | ВКР regulation, Appendix 1 |
| Conclusion | Conclusion should summarize results in relation to goals/tasks and may include practical proposals. | ВКР regulation, Appendix 1 |
| Paper format | A4, 210 x 297 mm. | ВКР regulation, Appendix 2 |
| Margins | Left 30 mm, right 15 mm, top 20 mm, bottom 20 mm. | ВКР regulation, Appendix 2 |
| Paragraph indent | Paragraph indent is 1.25 cm and consistent throughout the text. | ВКР regulation, Appendix 2 |
| Line spacing | 1.5 line spacing. | ВКР regulation, Appendix 2 |
| Font | Recommended font: Times New Roman. | ВКР regulation, Appendix 2 |
| Font size | Main text: 12-14 pt. Chapter/paragraph headings: 14 pt. Table text and figure/table captions: 12 or 14 pt. | ВКР regulation, Appendix 2 |
| Alignment | Main body text is justified. | ВКР regulation, Appendix 2 |
| Page numbering | Pages are numbered sequentially. Title page counts as page 1, but the number is not printed on the title page. Page numbers are centered in the bottom margin. | ВКР regulation, Appendix 2 |
| Heading numbering | Chapters/sections/subsections use Arabic numerals. Section numbering is hierarchical with periods. | ВКР regulation, Appendix 2 |
| Figure/table placement | Figures, schemes, diagrams, and tables are placed after the first reference to them, or on the next page. | ВКР regulation, Appendix 2 |
| Figure caption | Figure title is placed under the figure and centered; numbering uses Arabic numerals. The Russian template is `Рисунок N - Title`. | ВКР regulation, Appendix 2 |
| Table caption | Table title is placed above the table, left-aligned, without paragraph indent, in one line with the table number through a dash. | ВКР regulation, Appendix 2 |
| References | References are formatted according to GOST 7.1-2003 and GOST 7.82-2001. In-text bibliographic references follow GOST R 7.0.5-2008. | ВКР regulation, Appendix 1 and Appendix 2 |
| Reference coverage | Every source in the reference list must be reflected in the text. | ВКР regulation, Appendix 1 |
| Reference ordering | The PDF permits several ordering schemes: alphabetical, by document type, order of use, or chronological. | ВКР regulation, Appendix 1 |
| Appendices | Each appendix starts on a new page, is labeled at top center, has a centered bold title without a period, and must be referenced in the text. | ВКР regulation, Appendix 1 |
| Title page | A title page template is provided in Appendix 6. The PDF also says the title page is formed in the personal account when uploading the work. | ВКР regulation, Appendix 1 and Appendix 6 |
| School/department requirements | PhysTech schools and departments may set or clarify additional requirements for VKR preparation and defense. | ВКР regulation, section 2.17 |
| Printed page side | The text is prepared on one side of A4 sheets. | ВКР regulation, Appendix 2 |
| Defense presentation | The defense presentation should reflect the topic, relevance, goals/tasks, object/methods, essence of the work, and personal contribution. | ВКР regulation, section 3 |
| Defense materials deadline | Defense materials are submitted to the GEC secretary no later than 2 days before defense. | ВКР regulation, section 3 |
| Defense materials | Defense materials include the printed VKR, supervisor feedback, review where required by VKR type, plagiarism certificate, EBS placement permission, and other materials required by the department/secretary. | ВКР regulation, section 3 |
| Plagiarism check result | After upload, plagiarism checking is performed in Antiplagiat and a certificate/report is generated in the personal cabinet. | EBS order, sections 2.11-2.12 |
| EBS annotation access | Annotations are placed in open access except for works containing state secrets. | EBS order, section 2.9 |
| Storage | Electronic VKR versions are stored in EBS for at least 5 years. | EBS order, section 3.1 |

## Working Formatting Decisions for the Draft

These decisions stay inside the confirmed ranges above and should be used for the Markdown-to-DOCX conversion unless the supervisor or department gives a stricter template. The current VKR type is fixed as a bachelor thesis.

| Item | Working decision |
|---|---|
| VKR type | Bachelor thesis |
| Thesis language | English |
| Target length | 30-40 pages excluding appendices |
| Minimum originality target | At least 70% |
| Main font | Times New Roman |
| Main text size | 14 pt |
| Caption/table text size | 12 pt |
| Line spacing | 1.5 |
| Margins | left 30 mm, right 15 mm, top 20 mm, bottom 20 mm |
| Printed page side | One-sided printing |
| Paragraph indent | 1.25 cm |
| Body alignment | Justified |
| Page numbering | Bottom center; no visible number on title page |
| References order | Order of first use, because the PDF explicitly permits ordering by use |
| Figure captions in English draft | Use the equivalent English format `Figure N - Title`, centered below the figure. This is the English rendering of the confirmed Russian `Рисунок N - Title` format and should be confirmed with the supervisor before final submission. |
| Table captions in English draft | Use `Table N - Title` above the table, left-aligned, no paragraph indent. |
